## Tips

- Use high-quality images (but optimize them for web)
- Recommended tools for optimization:
  - https://tinypng.com/
  - https://squoosh.app/
  - ImageOptim (Mac app)
- Keep file sizes reasonable (under 500KB for most images)
- Use .jpg for photos, .png for graphics with transparency
